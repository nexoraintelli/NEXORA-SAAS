# Nexora SaaS Base

Estrutura inicial organizada para transformar a Nexora em SaaS.

## Fluxo principal

Index → Login → Dashboard → Cliente/Produto → Cérebro 1 → Cérebro 2 → Cérebro 3 → Síntese → Relatório

## Pastas

- `pages/`: páginas HTML do sistema
- `js/`: lógica separada por módulo
- `styles/`: CSS global
- `assets/`: imagens, logos e ícones
- `api/`: endpoints serverless
- `docs/`: documentação técnica e checklist

## Prioridade de implementação

1. Supabase único
2. Auth único
3. Cérebro 1 sem dependência de IA
4. Cérebro 2 com motor próprio
5. Cérebro 3 com cálculo próprio
6. Síntese com causa raiz e score real
7. Relatório profissional
8. Dashboard com histórico real
